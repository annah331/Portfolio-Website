Recipe Finder App

A web app to quickly find and save recipes using JavaScript and APIs

This project lets users search for recipes by ingredient or meal type. It uses the Edamam API to fetch data and displays results with a responsive interface.

Built with HTML, CSS, JavaScript, and a touch of localStorage to save favorite recipes. The focus was on clean design, fast load times, and easy navigation.
